## customer-shopping-behavior-analysis

## Project Overview

Analyze customer shopping behavior using Python, MySQL, and Power BI to uncover spending patterns, customer segmentation, subscription trends, and product performance. The project demonstrates a complete analytics workflow from data cleaning to interactive dashboard development.

## Tech Stack
- Python
- Pandas
- MySQL
- SQL
- Power BI
- Jupyter Notebook

## Dataset
- Records: 3,900
- Features: 18
- Customer demographics
- Purchase history
- Product categories
- Subscription status
- Discounts
- Ratings

## Project Workflow

```text
Dataset
   │
   ▼
Python (Data Cleaning)
   │
   ▼
MySQL (Business Analysis)
   │
   ▼
Power BI Dashboard
   │
   ▼
Business Insights
```

## Business Questions Solved
- Revenue by Gender
- Revenue by Age Group
- Sales by Category
- Customer Segmentation
- Subscriber vs Non-Subscriber Analysis
- Discount Impact
- Top Products
- Shipping Analysis 

## Key Insights
-Analyzed 3,900 customer transactions generating $233K+ in revenue.
-Male customers contributed 67.7% of total revenue.
-Clothing generated 44.7% of total revenue.
-Clothing and Accessories together contributed 76.5% of total revenue.
-Middle-aged customers generated 38.5% of business revenue.
-Average purchase amount was $59.76.
